# Slot Booking Web Application

## Overview

This is a Slot Booking Web Application developed using Flask and MySQL. Users can view available slots, book a slot, and view all booked slots.

## Features

- View available slots
- Book a slot
- Prevent duplicate bookings
- View all booked slots
- MySQL database integration
- Responsive UI using Bootstrap

## Technology Stack

- Python
- Flask
- SQLAlchemy
- MySQL
- Bootstrap
- Gunicorn
